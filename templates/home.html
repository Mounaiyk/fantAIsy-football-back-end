The exisiting routes on this server are:

  • "/": this home page
  • "/allstats" : returns stats of all players
  • "/stats/PLAYERNAME": returns stats of the individual player
  • "/teams/TEAMNAME": returns all stats of players for a certain team
  • "/getuserteam" : POST method, send fpl email, password and user id, returns players team
  • "/userplayer/PLAYER_ID" : returns stats of the individual player
